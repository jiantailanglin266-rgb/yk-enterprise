01 — Mission / Vision / Value

私たちの理念

Mission

テクノロジーと美意識で、
ビジネスの未来を再定義する。

優れた技術は、優れた美意識と出会ったとき、はじめて人を動かす力になる。私たちはAI・Web・マーケティング・美容の境界を越え、企業の可能性を最大限に引き出すことを使命としています。

Vision

AIと人の創造性が響き合う、
アジアを代表するクリエイティブDXカンパニーへ。

生成AIがあらゆる産業を塗り替える時代。私たちはテクノロジーを「人の代替」ではなく「創造性の増幅装置」と捉え、日本発のクリエイティブDXを世界へ届けます。

Value 01

成果に、誠実であれ。

納品はゴールではなくスタート。数字で語り、成果が出るまで伴走する。クライアントの事業成長こそが、私たちの唯一の評価軸です。

Value 02

最先端を、あたりまえに。

昨日の正解を疑い、今日の最先端を実装する。AIを全社員が使いこなし、学び続ける組織であることを、私たちは自らに課しています。

Value 03

美しさは、機能である。

美しいものは、伝わる。使いやすいものは、選ばれる。デザインを装飾ではなく事業の武器として磨き続けます。

02 — Message

代表メッセージ

代表取締役CEO

山田 恭平

Kyohei Yamada

「変化を恐れる企業」と「変化を武器にする企業」。
その差が、これほど残酷に開く時代はありません。

生成AIの登場は、インターネットの誕生に匹敵する地殻変動です。検索のかたちが変わり、ものづくりの速度が変わり、顧客との出会い方そのものが変わりました。しかし多くの企業にとって、この変化はまだ「遠いニュース」のままです。私たちYK ENTERPRISEは、その距離をゼロにするために存在しています。戦略から実装、運用までを一気通貫で担い、AIの力を「明日の売上」に変換する。それが私たちの仕事です。

同時に、私たちはテクノロジー万能主義には立ちません。どれほどAIが進化しても、最後に人の心を動かすのは、美しさであり、物語であり、細部への執念です。テクノロジーと美意識——この二つを兼ね備えたパートナーとして、お客様のビジネスの未来を共に描けることを、心から楽しみにしています。まずは一度、私たちに会いに来てください。

04 — History

沿革

創業から現在まで。変化を武器に、事業領域を広げ続けてきました。

  1. 2018年4月

    創業

    東京都渋谷区にて株式会社YKエンタープライズを設立。Web制作事業からスタート。

  2. 2020年7月

    事業拡大

    Webコンサルティング事業・SNSマーケティング事業を開始。プロジェクト実績100件を突破。

  3. 2023年5月

    AI開発事業を開始

    生成AIの本格普及を受け、AI開発事業部を新設。生成AI導入支援・RAG構築サービスを提供開始。

  4. 2024年10月

    美容事業を開始

    IT×美容の新領域「Beauty DX」に参入。サロンプロデュース・AI美容診断サービスを展開。

  5. 2026年4月

    現在

    プロジェクト実績300件超、AI導入支援80社超。LLMOコンサルティングを正式リリースし、5事業体制で成長を加速中。
